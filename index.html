XML-Extensible Markup Language. XML files are used to communicate data in a structured way because they are platform independent. It is a text format that allows data to be stored in a form that is easy to read by both machines and people. CSV-comma-separated values. Instead of storing information in columns, CSV files store information separated by commas. Text and numbers stored in a CSV file can easily be transferred from one program to another. JavaScript Object Notation, JSON–a lightweight computer data exchange format. JSON is a text format based on a subset of the JavaScript language. Data structured in JSON is much more readable. It is also valued for its relatively small file size and the ability to simplify complex data models.
